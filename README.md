# Node Knight Chess

This project was developed to assess Node + React skills developing a new project to identify Knight's available movements in two turns.

# Getting Started
- `docker-compose build`
- `docker-compose up`
- Access localhost:3000

# Tests
- `cd backend && npm test`
- `cd ../client && yarn test`
